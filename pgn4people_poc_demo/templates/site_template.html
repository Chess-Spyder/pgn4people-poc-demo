Welcome to pgn4people!

pgn4people is a demonstration, and proof of concept, of a new and better way to navigate complex chess games and repertoires. This web app allows you to play around with the pgn4people concept using a large, sample PGN file meant to simulate working with a large repertoire. If you like the pgn4people idea, please let the developers of your favorite chess apps and websites know!

Although this web app is not directly useful, itâ€™s meant to inspire developers to incorporate the pgn4people paradigm into their already awesome chess apps and websites, to make them that much better.

Read more about what pgn4people does, and how it improves on traditional PGN interfaces, in the README document for a GitHub repository for a related version of this software.
